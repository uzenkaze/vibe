📖 애플리케이션 사용 가이드

⚙️ JSON 데이터 관리 (백업 및 복원)

본 서비스는 브라우저의 로컬 스토리지에 데이터를 저장합니다. 데이터 유실 방지를 위해 주기적인 백업을 권장드립니다.

  • • 파일로 다운로드: 현재 등록된 모든 일감 데이터를 .json 파일로 즉시 저장합니다.
  • • JSON 복사하기: 데이터 구조를 텍스트 형식으로 클립보드에 복사합니다.
  • • 데이터 가져오기: 백업해둔 JSON 텍스트를 입력창에 붙여넣고 [가져오기 적용]을 누르면 즉시 복구됩니다. (기존 데이터는 덮어씌워집니다.)

⚙️ 데이터 관리 (백업 및 복원)

브라우저 저장소의 데이터를 관리하거나 외부 JSON 파일로 백업할 수 있습니다.

  • • 파일로 다운로드: 현재 일감을 .json 파일로 즉시 백업합니다.
  • • JSON 복사하기: 데이터를 텍스트 형식으로 클립보드에 복사합니다.
  • • 데이터 가져오기: 백업된 JSON을 입력하여 즉시 데이터를 복구합니다.

📋 데이터 구조 이해

"title": "업무 명칭"
"status": "todo, progress, done"
"work_no": "하이지스 업무번호"
"task_no": "포탈 일감번호"

✨ 주요 기능 활용 팁

🔍 빠른 검색 업무명뿐만 아니라 업무번호, 일감번호로도 즉시 필터링이 가능합니다.
📅 달력 활용 접수일 기준으로 점이 표시되며, 클릭 시 해당 일감의 수정창이 바로 열립니다.
📄 상세 사이드바 목록의 업무명을 클릭하면 우측에서 상세 정보가 슬라이드 형태로 나타납니다.
🌦 날씨 정보 상단 날씨 위젯을 클릭하면 전국 주요 도시의 현재 온도를 지도로 볼 수 있습니다.

⚙️ 데이터 관리 (로컬 스토리지)

로컬 저장소의 데이터를 백업하거나 다른 브라우저로 안전하게 옮길 수 있습니다.
